⚠️ Application Error

The app failed to load. This is usually due to a missing setting or a local browser error.


      
Try these steps:

      
    
        
  • Check if your .env file exists and has the API keys.
    • 
        
  • Run npm install in your terminal.
    • 
        
  • Restart the server (Ctrl+C then npm run dev).
